# GuideRoom

Create a live audio room in 30 seconds. Visitors scan and listen with their own phones.

GuideRoom is an MVP for small tour guides, schools, company visits, museums, and temporary group activities. The first version focuses only on the core loop:

> Create room → show QR code → visitors scan → listen to the guide's live voice.

## MVP scope

The first stage intentionally does not include AI translation, subtitles, native mobile apps, complex accounts, or payment.

## Planned stack

- Next.js
- TypeScript
- Tailwind CSS
- LiveKit for real-time audio
- QR code based room sharing

## Roadmap

1. Project initialization
2. Static page prototype
3. Room creation and state
4. Real-time audio integration
5. Mobile testing
6. Real user pilot test

## Local development

```bash
cd apps/web
npm install
npm run dev
```

Then open http://localhost:3000.
